## Здесь собраны ansible-роли для конфигурирования ОС и установки ПО
#### **/astra17** - роли для AstraLinux SE 1.7.2 (Смоленск)
* ***Astra-server*** - первичная настройка ОС: задаем имя машины, настраиваем репозитории, обновляем систему, конфигурируем сеть.
* ***ALD_Pro-server*** - установка первого контроллера домена ALD Pro и первичная его настройка.
* ***ALD_Pro-client*** - установка клиента ALD Pro и подключение к контроллеру домена.
* ***Bacula-server*** - установка сервера Bacula и первичная его настройка заданий.
* ***Bacula-client*** - установка клиента Bacula и подключение его к серверу.
* ***Zabbix-server*** - установка сервера Zabbix (v5.0) и первичная его настройка.
* ***Zabbix-client*** - установка клиента Zabbix (v5.0) и подключение его к серверу.

#### **/astra16** - роли для AstraLinux SE 1.6.10 (Смоленск)
* ***ALD-server*** - установка контроллера домена ALD и первичная его настройка.
* ***ALD-client*** - установка клиента ALD и подключение к контроллеру домена.
* ***PXE-server*** - установка сервера сетевой загрузки (PXE) и его настройка.
* ***Bacula-server*** - установка сервера Bacula и первичная его настройка заданий.
* ***Bacula-client*** - установка клиента Bacula и подключение его к серверу.
* ***Zabbix-server*** - установка сервера Zabbix (v3.4) и первичная его настройка.
* ***Zabbix-client*** - установка клиента Zabbix (v3.4) и подключение его к серверу.
